Agama - Alternate Data Location.

Read it completely before use.

Standard 'Agama data directory`:
Linux: `~/.Komodo`
Mac OS: `/Users//Library/Application Support/Komodo`
Windows 10: `%AppData%\Komodo`

Procedure:

To choose another location for your blockchain and wallet files in Verus Agama, you will need to tell Agama which existing
location to use.

If this is a completely fresh install of Agama and want to start with a custom location follow this procedure:
1)  Create an empty directory anywhere on a drive with at least 30GB free space for just Verus. If you plan on using other Blockchains
    in native mode, reserve more space accordingly.
2)  Start Agama and click `Quick access` on the welcome screen and select `settings`.
3)  Go to `App Config (config.json)` and scroll down to `Agama data directory`.
4)  Enter the full path to your new directory that you created in step 1.
5)  scroll down and click on `Save app config`, close the windows and exit Agama.
6)  Start Agama to start using the new location.

If you have an existing installation of Verus Agama and you wish to move your standard blockchain and wallet data folders
follow this procedure:
1)  Create an empty directory anywhere on a drive with at least 30GB free space for just Verus. If you plan on using other Blockchains
    in native mode, reserve more space accordingly.

Before proceeding with step 2, make sure you have a reliable backup of your `wallet.dat` and/or the WIFs of your addresses.

2)  Make sure Agama is closed and move the contents of the standard `Agama data directory` (location is mentioned above the procedures
    on this page) to the location you created in step 1.
3)  Start Agama and click `Quick access` on the welcome screen and select `settings`.
4)  Go to `App Config (config.json)` and scroll down to `Komodo data directory`.
5)  Enter the full path to your new directory that you created in step 1.
6)  scroll down and click on `Save app config`, close the windows and exit Agama.
7)  Start Verus Agama as normal.

PS1. In future any reference to `VRSC Wallet Location` will be the directory you created in step 1.
PS2. The data location will be changed for ALL native chains you have now or use later.
